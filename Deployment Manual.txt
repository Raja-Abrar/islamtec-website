2. Create a GitHub account (if you don‚Äôt have one)

PART 2 ‚Äî Push Project to GitHub
3. Initialize Git in your project folder

git commit -m "Initial Islamtec website"

4. Create a new GitHub repository

Go to GitHub ‚Üí New Repository

Repository name:

islamtec-website


Visibility: Public (recommended for easy sharing)

Do NOT add README, .gitignore, or license

Click Create Repository

5. Connect local project to GitHub

GitHub will show commands. Use these:

git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/islamtec-website.git
git push -u origin main


(Replace YOUR_USERNAME)

PART 3 ‚Äî Deploy on Vercel
6. Create Vercel account

Go to vercel.com

Sign up using GitHub

Authorize Vercel access

7. Import project

Click New Project

Select islamtec-website repo

Click Import

Vercel will auto-detect:

Framework: Next.js

Build Command: next build

Output: .next

Do NOT change anything yet.

PART 4 ‚Äî Environment Variables (IMPORTANT)
8. Add .env.local values

If your project uses email/WhatsApp/contact API:

In Vercel ‚Üí Project ‚Üí Settings ‚Üí Environment Variables

Add exactly what you have in .env.local, for example:

Environment: Production

Click Save

PART 5 ‚Äî Deploy
9. Deploy

Click Deploy

Wait ~1 minute.

You will get a live URL like:

https://islamtec-website.vercel.app
